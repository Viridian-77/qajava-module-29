Найдены следующие ошибки:
-1, 0, 1, 10, 11, 12 лет - не тинейджер, но система говорит, что это тинейджер
19 лет - тинейджер, но система говорит, что это не тинейджер

В методе isTeenager() следующие ошибки:
1. Не установлена нижняя граница возраста
2. Верхняя граница возраста должна включать 19
Метод должен быть написан так:
    public static boolean isTeenager1(int age) {
        boolean result;
        if (age <= 19 && age >= 13) result = true;
        else result = false;
        return result;
    }
